# Exploratory-Data-Analysis
This project focuses on performing Exploratory Data Analysis (EDA) using Python libraries such as Pandas, Matplotlib, and Seaborn.
The goal is to analyze the dataset, visualize patterns, identify relationships between variables, and extract meaningful insights from raw data.

Dataset used:

Iris Dataset / Tips Dataset
